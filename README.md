# RblxAIPlayer
This project is an open source developer showcase of how an AI "Player" interacts with a 3D world. Every script in this repository is made for reproductivity so you can easily integrate this into your game.

## Getting Started
Git clone this repository for the scripts, and if you want the demo game you can get it from releases or by going to it's Roblox page
